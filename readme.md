# MICROFRONTENDS
## Module Federation

### Run the project

The main project is mfe2 and mfe1 is the sub.<br/>
Run the sub at first
```
cd mfe1
webpack serve
```

Then

```
cd mfe2
webpack serve
```

* Or change the scripts in the **package.json** file in both projects
